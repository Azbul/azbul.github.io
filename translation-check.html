Machine translation detector

Детектор машинного перевода

Приложение определяет схожесть пользовательского перевода с переводом от Гугла, Яндекса, а также переводчиков translate.ru, reverso.net, deepl.com и bing.com.

  • Backend: Console App
  • Client: WPF, Telegram Bot
  • DB: PostgreSQL
  • Other: Entity Framework, EPPlus, Selenium, PhantomJS, HTTP Web classes
Клиент (WPF)

Приложение берет оригинальный текст и прогоняет ее по переводчику, полученный перевод сравнивает с пользовательским переводом по алгоритму шингла, и так проходится по каждому сервису-переводчику (по очереди или параллельно/многопоточно, если ресурсы системы позволяют).

Полезное

Необходимо было реализовать асинхронный запуск задачи со STA потоком. Привычный асинхронный вызов оказался не рабочим (т.к. асинхронный вызов заворачивается в новый таск, который не является STA). Решение - запуск потока в асинхронной задаче, установив потоку ApartmentState.STA:

private async void CreateStaTaskAsync()

{

await Task.Run(() =>

            {

                Thread worker = new Thread(

                    delegate ()

                    {

                        // наш код

                    });

                worker.SetApartmentState(ApartmentState.STA);

                worker.Start();

                worker.Join();

            });

}
